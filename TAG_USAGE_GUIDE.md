# 书签标签功能使用指南

## 功能概述

我为你的书签系统添加了完整的标签功能，现在每个书签卡片都可以显示自定义标签，并支持通过标签进行搜索和筛选。

## 数据格式

在YAML文件中，为每个书签添加`tags`字段：

```yaml
- name: DragGAN
  url: https://github.com/XingangPan/DragGAN
  img: /images/logos/github.png
  description: 通过拖动修改图像，例如更改表情，面朝方向等
  tags: ["AI编辑", "开源", "GitHub"]

- name: tinypng
  url: https://tinypng.com/
  img: /images/favicon.ico
  description: 在线图片压缩工具
  tags: ["图片压缩", "在线工具", "PNG", "JPG"]
```

## 标签显示规则

- **最多显示3个标签**：如果标签超过3个，会显示前3个，剩余的用"+数字"表示
- **响应式设计**：在不同屏幕尺寸下自动调整标签大小
- **夜间模式适配**：标签在夜间模式下有专门的配色方案

## 交互功能

### 1. 标签点击搜索
- 点击任意标签，会自动在搜索框中输入该标签内容
- 页面会实时筛选出包含该标签的所有书签

### 2. 搜索增强
- 搜索功能现在支持：
  - 书签名称搜索
  - 描述内容搜索
  - **标签搜索**（新增）

### 3. 视觉效果
- 标签有悬停效果，鼠标悬停时会放大
- 点击标签时有视觉反馈
- 标签颜色与整体设计风格一致

## 推荐的标签分类

### 按功能分类
- `图片编辑`、`图片压缩`、`图片放大`、`背景移除`
- `格式转换`、`图片搜索`、`OCR识别`

### 按技术特征
- `AI`、`开源`、`在线工具`、`自部署`
- `批量处理`、`实时处理`、`专业`

### 按平台/来源
- `GitHub`、`Google`、`商业`、`免费`

### 按文件格式
- `PNG`、`JPG`、`SVG`、`PSD`

## 使用建议

1. **保持标签简洁**：每个标签建议2-4个字符
2. **标签一致性**：相同功能的工具使用相同的标签名称
3. **适量标签**：每个书签建议2-5个标签，避免过多
4. **中文优先**：使用中文标签更符合用户习惯

## 示例标签组合

```yaml
# AI图片工具
tags: ["AI编辑", "开源", "GitHub"]

# 在线压缩工具  
tags: ["图片压缩", "在线工具", "PNG", "JPG"]
```
